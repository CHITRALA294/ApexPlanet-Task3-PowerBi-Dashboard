ApexPlanet Task 3 – Power BI Dashboard

Objective

To create an interactive dashboard to visualize sales performance and identify key business insights.

Dashboard Features

• Region-wise sales analysis
• Segment-wise sales comparison
• Monthly sales trend visualization
• Interactive filters and visual insights

Tools Used

• Power BI
• Data Visualization
• Data Analysis

Insights

- The West region generated the highest sales.
- Consumer segment contributed the most revenue.
- Sales show noticeable growth trends across months.

This dashboard demonstrates how data visualization can support better business decision-making.
